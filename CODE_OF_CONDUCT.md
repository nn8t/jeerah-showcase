# Jeerah Code of Conduct

> Community and professional interaction standards for the public Jeerah showcase repository.

---

## Purpose

This repository exists to present Jeerah as a professional software product showcase.

All discussions, issues, comments, and feedback should remain respectful, constructive, and relevant to the public documentation.

---

## Expected Behavior

Participants are expected to:

- Communicate respectfully
- Stay on topic
- Provide constructive feedback
- Respect the project's closed-source status
- Avoid requesting private implementation details
- Avoid sharing sensitive information publicly
- Respect intellectual property rights

---

## Unacceptable Behavior

The following behavior is not acceptable:

- Harassment or personal attacks
- Spam or irrelevant promotion
- Attempts to obtain private source code
- Attempts to pressure the owner into disclosing secrets
- Public posting of suspected vulnerabilities
- Reverse engineering requests
- Copying or claiming project ownership
- Sharing private user or business data
- Abusive or discriminatory language

---

## Security and Privacy

Do not post sensitive information publicly.

This includes:

- API keys
- Environment variables
- Production URLs
- Database details
- Customer data
- Driver data
- Payment details
- Security findings
- Private screenshots
- Internal implementation assumptions

---

## Enforcement

The project owner may remove comments, close issues, restrict participation, or block users who violate this code of conduct.

---

## Scope

This code of conduct applies to all interactions related to the public Jeerah showcase repository.

---

## Summary

Professional discussion is welcome.

Attempts to misuse, copy, pressure, or expose private parts of Jeerah are not.
